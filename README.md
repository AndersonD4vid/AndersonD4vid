# Sobre mim
Olá, meu nome é Anderson!

Sou um desenvolvedor de software com experiência em grandes projetos, atuando desde o planejamento até o desenvolvimento final. Comecei minha jornada na tecnologia em 2016 e me especializei no início de 2020.

Meu primeiro contato com a programação foi em 2016, quando comecei a explorar HTML, CSS e JavaScript. Naquela época, eu me divertia criando lojas integradas com botões de pagamento do PayPal, o que despertou meu interesse pelo desenvolvimento web e mobile.

Minha trajetória no desenvolvimento de software tem sido cheia de desafios e aprendizados. Agradeço a todos que acompanharam minha história e espero poder compartilhar meu conhecimento com outros, além de continuar aprendendo com vocês.
